## 3. Description（コピペ用）

```
# 囲Trap (I-Trap)

**Go meets Tetris!** A falling-block puzzle where you use the rules of the ancient board game Go to clear stones.

## How to Play

1. **Drop** black and white stones onto the board
2. **Surround** a group of stones to capture them (just like in Go!)
3. **Chain** multiple captures for massive combo scores
4. No Go knowledge needed — the tutorial teaches you in 30 seconds

## Controls

- **← → Arrow keys**: Move piece
- **↑ Arrow / Z**: Rotate
- **↓ Arrow**: Soft drop
- **Space**: Hard drop
- **Touch controls** available on mobile

## Features

- 🎵 Dynamic BGM that shifts when danger rises
- 🏆 Online leaderboards (daily / weekly / monthly)
- 🌐 English & Japanese
- 📱 Mobile-friendly with touch controls
- 🎮 Works offline as a PWA
```
